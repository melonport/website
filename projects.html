Our Projects

Melon mail is a state-of-the-art end-to-end encrypted, secure messaging service between managers and investors, so you can safely exchange information without worrying about where your data goes. The application is 100% decentralized, since it consists entirely of the front-end code interacting with Ethereum smart contracts and storing data on IPFS.

Oyente is an open source analysis and formal verification tool for Ethereum smart contracts. It can be used to detect many common bugs found in smart contacts like r-eentrancy, transaction ordering dependence and so on. What’s more interesting is that Oyente’s design is modularized, so this allows advanced users to implement and plug in their own detection logic to check self-defined properties in their contracts.
